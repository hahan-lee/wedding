## 깃사용법

    - 기본 리눅스 명령어 익히기
    ls : 파일, 폴더(디렉토리) 목록
    ls -la : 숨김 목록까지 모두 보이기
    - 현재 배포할 파일 목록을 확인

```js
ls - la;
```

## 깃설정

```js
    ls -la
    git init
    ls -la
```
